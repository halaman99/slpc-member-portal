## ⚡ Deploy in 5 Steps (~10 minutes)

### See: PHASE1_IMPLEMENTATION.md for detailed instructions

**Step 1:** Database Migration
- Copy SUPABASE_MIGRATION_PHASE1.sql
- Paste in Supabase SQL Editor
- Click Run

**Step 2:** Enable Realtime
- Supabase → Replication
- Toggle ON: members, duties, events, announcements, admin_invites

**Step 3:** Set First Admin
```sql
UPDATE members 
SET is_admin = TRUE, admin_approved_by = id 
WHERE email = 'smigoal.lul@gmail.com';
```

**Step 4:** Delete Sample Data
```sql
DELETE FROM announcements;
DELETE FROM member_formations;
DELETE FROM formations;
DELETE FROM events;
```

**Step 5:** Deploy Code
```bash
cd d:\SLPC-MAS\app

git commit -m "Phase 1: Admin system + realtime"

```

## 🎯 What You Can Do Now

### Regular Members:
```typescript
// Get live member list (auto-updates!)
const { members } = useRealtimeMembers()

// See live duties
const { duties } = useRealtimeDuties()

// Accept admin invite
await acceptAdminInvite(inviteId)
```

### Admins:
```typescript
// Manage all members
await inviteAdmin('user@email.com')
await updateMember(memberId, { role: 'Secretary' })

// Track changes
SELECT * FROM audit_log

// Check admin count
const count = await getAdminCount()
```
